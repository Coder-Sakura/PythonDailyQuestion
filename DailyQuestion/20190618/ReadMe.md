#### 基础题：

```
列表 languages = [(1, 'Go'), (2, 'Ruby'), (3, 'Swift'), (4, 'Erlang'), (5, 'Kotlin'), (6, 'Python')]，
请进行排序，实现效果：
[(4, 'Erlang'), (5, 'Kotlin'), (6, 'Python'), (3, 'Swift'), (2, 'Ruby'), (1, 'Go')]
```
